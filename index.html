    Supervisor add-ons

    I run a Supervised install with the following add-ons:

    • Cloudflared 2.0.7
    • MariaDB 2.5.1
    • Mosquitto broker 6.1.2
    • Rclone 3.0.10
    • Studio Code Server 5.2.1
    • Traefik 4.1.4
    • Zigbee2MQTT 1.27.0-1
